# 🚗 Car Price Prediction

## 📌 Project Description
This is a Machine Learning project that predicts the selling price of a car using Linear Regression. The model is trained on an automobile dataset and can estimate the selling price based on different car features.

## 🛠️ Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn

## 📂 Dataset
- automobile_dataset.csv

## 📈 Machine Learning Model
- Linear Regression

## ✨ Features
- Data preprocessing
- Model training
- Price prediction
- Mean Squared Error (MSE) evaluation

## ▶️ How to Run
1. Open `Car_Price_Prediction.ipynb` in Google Colab or Jupyter Notebook.
2. Upload `automobile_dataset.csv`.
3. Run all cells.
4. Enter new car details to predict the selling price.

## 📊 Sample Output
Predicted Selling Price = $25709.25
